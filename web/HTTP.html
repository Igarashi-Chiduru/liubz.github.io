# HTTP

  1. content-type: image/jpeg 显示类型
  2. content-length: 12324 文件大小
  3. 常见状态码
  • 200 正常
  • 301 表示旧地址A的资源已经被永久地移除了
  • 302 重定向，到其它地方获取资源
  • 304 缓存
  • 404 没有找
  1. 报文

  1. 基本的浏览器连接处理 (a) 浏览器从 URL 中解析出服务器的主机名； (b) 浏览器将服务器的主机名转换成服务器的 IP 地址； (c) 浏览器将端口号（如果有的话）从 URL 中解析出来； (d) 浏览器建立一条与 Web 服务器的 TCP 连接； (e) 浏览器向服务器发送一条 HTTP 请求报文； (f) 服务器向浏览器回送一条 HTTP 响应报文； (g) 关闭连接，浏览器显示文档

  2. 三次握手 详解 TCP 连接的“ 三次握手 ”与“ 四次挥手 ”
